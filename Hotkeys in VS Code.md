# Hotkeys in VS Code
#resource 

- CTRL + ` - open terminal
- CTRL + 1 - open editor
- CTRL + Tab - switch between open tabs
- CTRL + Shift + P - open command palette
- CTRL + P - go to file
- CTRL + , - open settings
- CTRL + K S - open hotkeys
- CTRL + F4 - close editor/window
- CTRL + J - move down/next
- CTRL + K - move up/previous
